# Car Dodger

Car Dodger is a simple 2D game where the player drives a car and tries to avoid other cars coming from the top of the road.  
The longer the player survives without crashing, the higher the score.

---

## Game Description

The game is based on quick reactions and focus.  
Enemy cars move downward on the road, and the player must move left or right to avoid them.  
If the player collides with an enemy car, the game ends.

---

## How to Play

-   **Move Left:** Press the left arrow key
-   **Move Right:** Press the right arrow key
-   **Exit Game:** Press the Escape key
-   **Goal:** Stay alive as long as possible

---

## Features

-   Simple and fun gameplay
-   Moving cars and smooth control
-   Score increases over time
-   Game becomes harder as you play longer
